# build_file

# step
1. download build_file
2. 修改CP_CefView.pro文件中ThirdLib_PATH = D:/qtfile/QtCef/src 成你的本地地址
3. qmark CP_CefView
4. build CP_CefView 生成CP_CefView.dll文件

# use 
1. 构建测试项目TestView
2. 在生成的TestView.exe目录中添加CP_CefView.dll 和 src/cef/补充文件cef 里的文件内容
3. 运行TestView项目
